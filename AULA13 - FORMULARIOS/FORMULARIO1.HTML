MEU PRIMEIRO FORMULARIO

FORMULARIO SEM LABEL

Essa é a TAG multifunção do HTML que compõe boa parte do formulário. Alterando o valor do atributo type, podemos escolher a função do input. O Atributo name identifica os campos do input, ele é o índice para o envio das informações. *Sem esse atributo, os dados não são enviados.

FORMULARIO COM LABEL

Por padrão, cada campo terá seu próprio

TAG "TEXTAREA"

"SELECT" Essa TAG abre a possibilidade de opções para o usuário. Como nos outros campos, necessita do parâmetro name para enviar os dados.

botao1

Um botão de tipo reset reinicia o formulário a seu estado inicial. Um botão de tipo submit tem a função de enviar os dados do formulário. Um botão de tipo button não realiza nenhuma ação, porém, podemos configurá-lo no JavaScript.
